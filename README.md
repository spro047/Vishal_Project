Follow these steps to view the project locally:

   ```bash
   git clone https://github.com/spro047/Vishal_Project.git
   cd Vishal_Project
   ```

2. **Open directly in a browser:**
   - Navigate into `frontend` folder and open `index.html` with any modern web browser.

3. **Run using a simple HTTP server** (recommended for proper asset loading):
   ```bash
   cd /home/shashank/Desktop/vishal_Project
   python3 -m http.server 8000
   ```
   Then open your browser at: `http://localhost:8000/frontend/`

   Alternatively, use Node.js if installed:
   ```bash
   npx http-server frontend
   ```
